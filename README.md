✨ 核心功能与“连招”说明

忘掉那些桌面上显眼的“隐藏应用”图标吧，真正的刺客总是悄无声息。 本模块监听物理按键，只有输入正确的“终极连招”，才会触发隐藏（冻结）机制。

🎮 触发连招操作指南：

1.  起手式：同时按住 音量加键 (Vol+) 和 电源键 (Power) 不放。
2.  蓄力：保持 音量加键 按住不动，松开 电源键。
3.  释放大招：在松开电源键后的 3秒内，再次按下 电源键。

✅ 触发成功：手机会微微震动一下（300ms），仿佛在对你说：“兄弟，稳了”。此时，你在名单里配置的 App
会瞬间被拔网线+冻结（从桌面消失，接收不到任何消息）。

💡 贴心设计： 如果连招搓错了（比如先松开了音量键，或者超过了 3 秒超时），什么都不会发生，状态机自动重置，深藏功与名。

🛠️ 食用方法

1.  安装本 Magisk 模块并重启设备。
2.  找到模块目录下的 apps.txt 文件。
3.  在里面填入你想“防查”的 App 的包名（一行一个）。
      - 示例
        cn.soulapp.android
        com.p1.mobile.putong
        com.tangxin.vlog
4.  保存，遇到长辈查岗或对象突击时，行云流水地搓出按键连招即可。

(注：解冻应用目前需要各位自己使用终端命令/或者用其他解冻工具恢复，毕竟本模块只管“杀”不管“埋”，保命第一！)
